ans: In a gitignore file, a programmer can create some ignore rules shared with other teammates, so GitHub knows with files (or pattern) it should ignore. What does fast-forward in Git mean?
ans: The fast-forward happens when a user merges branches. If master has no divergence, git just moving pointer forward to latest commit instead of creating a new commit.
