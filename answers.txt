What is a gitignore file and what should it include for a Java project?

Answers: In a gitignore file, a programmer can create some ignore rules shared with other teammates, so GitHub knows with files (or pattern) it should ignore. For a java project, it includes compiled class file, log file, BlueJ files, Mobile Tools for java, package files, and virtual machine crash logs.

What does fast-forward in Git mean?

Answers: The fast-forward happens when a user merges branches. If master has no divergence, git just moving pointer forward to latest commit instead of creating a new commit.
